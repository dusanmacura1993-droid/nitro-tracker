# nitro-tracker
Svi inputi se čuvaju u Firebase → svi korisnici vide live update

Dugme Add person dodaje novog igrača u tabelu i u bazu

Dugme Next Day resetuje vrednosti svih igrača

Progress bar i grafikon se automatski update-uju

Dugme Export (Discord) kopira tekstualni izveštaj sa trenutnim progresom
